# Protect the Core
Protect the Core event for Rift SMP

## Commands

**ALL COMMANDS REQUIRE OP (Except for rules)**\
/protectthecore OR /ptc: Main prefix for commands\
-> config
- Opens the main config menu for the plugin.
- Note: This isn't all the configuration that can be made with this plugin.

-> spawn_end_shop
- Spawns the end shop (The custom Protect The Core End dimension must be loaded!)

-> supply_drops
* Opens the supply drops config menu

-> event_time\
 -> \
-> rules
- Lists the rules for the event

-> give_crown\
-> give_core\
-> teams
* Opens the team config menu

-> start
-> stop
-> lower_wall
-> teleport_to\
